Research & Development Projects

Currently, our group is working on the following projects:

PKI for the Post-Quantum Era

This research project aims to propose strategies, integrate and evaluate Post-Quantum Cryptography in PKIs and Network Protocols

Post-Quantum Cryptography in CZERTAINLY open source platform

Establish (Hybrid) Post-Quantum Cryptography in CZERTAINLY open source plaftorm.

AVACPQ-R

This research aims to improve learning of Post-Quantum Cryptography: Lattice-based Cryptography and Cryptanalysis. Test here (Soon!).

Towards the search for security vulnerabilities in regional and open source community systems

Not all companies and institutions have enough resources and staff to ensure the security of their systems. This extension project is not necessarily related to PQC but aims to improve the security of systems used in the local (or regional) community.
